# quickd

#### 介绍
QuickD是一个前后端分离快速开发平台，是基于 Spring Boot 和 Vue 开发，整合 Flowable工作流、Shiro、Redis等，来帮助中小型企业及个人实现敏捷化的应用交付和运营管理，并提供代码生成器、通用前端等业务组件，来帮助开发者聚焦于业务，加速中小型企业数字化转型。
